# Чек-лист визуальной самооценки износа здания медицинской организации — PWA v44

## Структура файлов

Версия v44 возвращает прежнюю структуру GitHub: все основные файлы лежат в корне репозитория.

```text
PRECHECK.md
README.md
icon-192.png
icon-512.png
index.html
manifest.webmanifest
service-worker.js
```

Папка `icons/` не требуется.

## Что включено из v43

- кадастровый номер;
- координаты объекта;
- ссылка на точку в Яндекс.Картах;
- общее фото здания;
- усиленный первый блок отчёта «Что делать по результатам осмотра»;
- конкретные рекомендации;
- подсказки `?` по блокам оценки;
- понятные формулировки вместо «вес/вклад»;
- уточнения по кровле, протечкам, пожарной безопасности и санитарному состоянию.

## Обновление GitHub Pages

1. Распаковать архив v44.
2. Загрузить файлы в корень репозитория с заменой:
   - `index.html`
   - `manifest.webmanifest`
   - `service-worker.js`
   - `README.md`
   - `PRECHECK.md`
   - `icon-192.png`
   - `icon-512.png`
3. Нажать `Commit changes`.
4. Открыть сайт с параметром `?v=44`.
