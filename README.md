# PWA v26 — стабильный откат на рабочую базу v19

Версия собрана на базе рабочей версии v19.

## Восстановлено
- расчет предварительного индекса;
- отображение всех блоков формы;
- загрузка фото через камеру и галерею;
- комплексный отчет;
- печать / сохранение PDF через системное меню.

## Безопасные текстовые изменения
- «Красные флаги» заменены на «Критические признаки риска»;
- расчетный санитарный блок назван без лишней медицинской специфики.

## Обновление GitHub Pages
1. Загрузить содержимое архива с заменой файлов.
2. Нажать Commit changes.
3. Открыть сайт с параметром ?v=26.
4. На iPhone удалить старую PWA-иконку и добавить сайт заново.
