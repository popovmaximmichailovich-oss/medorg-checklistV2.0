# Чек-лист визуальной самооценки износа здания медицинской организации — PWA v45 stable rollback

## Что это

v45 — аварийный стабильный откат от рабочей базы v42.

## Почему сделан откат

Версии v43/v44 добавляли новые функции поверх рабочей базы и нарушили штатную работу приложения. В v45 спорные надстройки убраны, сохранена стабильная логика.

## Структура файлов

```text
PRECHECK.md
README.md
icon-192.png
icon-512.png
index.html
manifest.webmanifest
service-worker.js
```

## Сохранено

- расчёт предварительного индекса;
- мастер осмотра;
- критические признаки риска;
- отчёт;
- PDF/печать;
- фото;
- нижняя навигация;
- чистая шапка;
- прежняя структура GitHub без папки `icons/`.

## Обновление GitHub Pages

1. Распаковать архив.
2. Загрузить все файлы в корень репозитория с заменой.
3. Нажать `Commit changes`.
4. Открыть сайт с параметром `?v=45`.
5. На iPhone удалить старую PWA-иконку и добавить сайт заново.
