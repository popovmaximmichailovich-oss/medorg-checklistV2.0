## Голосовой ввод
Для iPhone оптимально использовать системную диктовку клавиатуры: нажать в текстовое поле → микрофон на клавиатуре → продиктовать комментарий. Встроенную браузерную транскрибацию лучше делать отдельным экспериментальным режимом, потому что на iOS она поддерживается нестабильно.

## Обновление GitHub Pages
1. Распаковать архив.
2. Загрузить содержимое архива в GitHub: index.html, manifest.webmanifest, service-worker.js, README.md, icons/.
3. Commit changes.
4. Открыть сайт с параметром ?v=25.
5. На iPhone удалить старую PWA-иконку и добавить сайт заново.
