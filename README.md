## Обновление GitHub Pages
1. Распаковать архив.
2. Загрузить содержимое архива через Code → Add file → Upload files.
3. Нажать Commit changes.
4. Открыть сайт с параметром ?v=23.
5. На iPhone удалить старую PWA-иконку и добавить сайт заново.
