# PWA v39 — bottom navigation

## Добавлено
- В нижнюю компактную панель добавлены кнопки:
  - Назад — переход к предыдущему блоку;
  - Далее — переход к следующему блоку;
  - Вверх — переход к началу текущего блока.
- Нижний статус v38 сохранён.
- Расчётная модель, отчёты, фото и PDF не менялись.

## Обновление
Загрузить файлы на GitHub с заменой и открыть сайт с параметром ?v=39.
