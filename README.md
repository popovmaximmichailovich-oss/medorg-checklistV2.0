# PWA v22 — Чек-лист визуальной самооценки износа здания медицинской организации

## Основные изменения v22
- Пошаговый мастер осмотра.
- Понятные варианты выбора вместо голых процентов.
- Несколько учреждений и несколько зданий/корпусов.
- Отчёт по каждому зданию.
- Сводный отчёт по учреждению.
- Показатель качества данных.
- Карточки дефектов.
- Экспорт CSV для реестра.
- Резервное сохранение JSON.
- Фото через камеру и галерею.
- Печатная PDF-верстка.
- Ограничение: приложение не является техническим обследованием по ГОСТ 31937-2024.

## Как обновить GitHub Pages
1. Распаковать архив.
2. Открыть репозиторий GitHub → Code → Add file → Upload files.
3. Загрузить содержимое архива с заменой файлов:
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - README.md
   - icons/icon-192.png
   - icons/icon-512.png
4. Нажать Commit changes.
5. Открыть сайт с параметром ?v=22.
6. На iPhone удалить старую иконку PWA и добавить сайт на экран Домой заново.
